# RNK Core MapGen Release Shell

This repository is a lightweight distribution shell for the core tier. Use it only for the release manifest, the release zip, and plain release notes.

## What ships
- `module.json`
- `module.zip`
- Tier-specific release notes

## Tier indicators
- `[INCLUDED]` Free-tier base map generation
- `[INCLUDED]` Theme-driven textures
- `[INCLUDED]` Compendium mob and loot spawning
- `[INCLUDED]` Trap controls
- `[INCLUDED]` Architecture controls
- `[INCLUDED]` Extended dungeon generation controls
- `[INCLUDED]` Patreon login requirement
- `[LOCKED]` Architect-tier widest layout controls

## Access model
- Core tier inherits the free and alpha feature sets.
- Authentication is required.
- Access is granted only to the core Patreon tier.

## Install path
Point Foundry to this repo’s manifest URL.
